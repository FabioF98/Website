## ğŸ’¡ Key Features Explained

### Custom Cursor

The site features a custom cursor that follows the mouse on desktop devices. It automatically disables on touch devices for better UX.

### Intersection Observer

Sections fade in smoothly as you scroll, using the Intersection Observer API for optimal performance (no scroll event listeners!).

### Form Validation

The contact form includes:
- Real-time validation as you type
- Custom error messages
- HTML5 validation with custom styling
- Accessibility features (ARIA labels, focus management)
